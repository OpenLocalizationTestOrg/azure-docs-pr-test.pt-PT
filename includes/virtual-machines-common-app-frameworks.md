## <a name="azure-cli-20"></a>CLI 2.0 do Azure
Estes passos exigem Olá CLI do Azure versão 2.0.4 ou posterior. Executar `az --version` versão de Olá toofind. Se precisar de tooupgrade, consulte [instalar o Azure CLI 2.0]( /cli/azure/install-azure-cli).

Em primeiro lugar, crie um grupo de recursos com [criar grupo az](/cli/azure/group#create). Em seguida, criar a implementação a partir de um modelo com [criar a implementação do grupo az](/cli/azure/group/deployment#create). Fornecer o modelo de tooyour URI completo Olá Olá `--template-uri` parâmetro e quaisquer necessários parâmetros de modelo no `--parameters` da seguinte forma:

```azurecli
az group create --name myResourceGroup --location eastus
az group deployment create --resource-group myResourceGroup \
    --template-uri https://raw.githubusercontent.com/azure/azure-quickstart-templates/master/{template-name}/azuredeploy.json \
    --parameters '{"sshKeyData": {"value": "ssh-rsa AAAAB3N{snip}B9eIgoZ"}}'
```

Dependendo do modelo de Olá, pode demorar algum tempo para recursos de Olá toodeploy do Azure.


## <a name="azure-powershell"></a>Azure PowerShell
Estes passos exigem Olá Azure PowerShell versão do módulo 3,6 ou posterior. Executar `Get-Module -ListAvailable AzureRM` versão de Olá toofind. Se precisar de tooupgrade, consulte [módulo Azure PowerShell instalar](/powershell/azure/install-azurerm-ps).

Em primeiro lugar, crie um grupo de recursos com [New-AzureRmResourceGroup](/powershell/module/azurerm.resources/new-azurermresourcegroup). Em seguida, criar a implementação a partir de um modelo com [New-AzureRmResourceGroupDeployment](/powershell/module/azurerm.resources/new-azurermresourcegroupdeployment). Fornecer o modelo de tooyour URI completo Olá Olá `-TemplateURI` parâmetro da seguinte forma:

```powershell
New-AzureRmResourceGroup -Name "myResourceGroup" -Location "EastUS"
New-AzureRmResourceGroupDeployment -ResourceGroupName "myResourceGroup" `
     -TemplateUri "https://raw.githubusercontent.com/azure/azure-quickstart-templates/master/{template-name}/azuredeploy.json"
```

Quando executa Olá [New-AzureRmResourceGroupDeployment](/powershell/module/azurerm.resources/new-azurermresourcegroupdeployment) comando, poderá ser pedido tooenter valores para parâmetros de Olá no modelo de Olá. Dependendo do modelo de Olá, pode demorar algum tempo para recursos de Olá toodeploy do Azure.
