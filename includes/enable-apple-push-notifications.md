
* [Registar um ID de aplicação para a sua aplicação](https://developer.apple.com/library/ios/documentation/IDEs/Conceptual/AppDistributionGuide/MaintainingProfiles/MaintainingProfiles.html#//apple_ref/doc/uid/TP40012582-CH30-SW991). Criar uma ID da aplicação explícito (não um ID de aplicação de carateres universais) e, para **ID do pacote**, utilizar ID de pacote exato Olá que esteja no seu projeto de início rápido do Xcode. Também é fundamental que selecionou Olá **notificações Push** opção. 
* Em seguida, [tooprepare para configurar as notificações push](https://developer.apple.com/library/ios/documentation/IDEs/Conceptual/AppDistributionGuide/AddingCapabilities/AddingCapabilities.html#//apple_ref/doc/uid/TP40012582-CH26-SW6), crie um "Desenvolvimento" ou "Distribuição" certificado SSL.
