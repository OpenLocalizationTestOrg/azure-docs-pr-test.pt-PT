# <a name="web-apps-documentation"></a>Documentação sobre as Aplicações Web

As Aplicações Web do Azure permitem-lhe criar e alojar aplicações Web na linguagem de programação da sua preferência sem ter de gerir infraestrutura. Este serviço garante dimensionamento automático e elevada disponibilidade, suporta Windows e Linux e permite implementações automáticas a partir do GitHub, Visual Studio Team Services ou de qualquer repositório de Git. Saiba como utilizar as Aplicações Web do Azure com os nossos inícios rápidos, tutoriais e amostras.
