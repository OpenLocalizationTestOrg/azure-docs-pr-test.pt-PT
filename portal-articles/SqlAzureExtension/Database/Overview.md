# <a name="sql-database-documentation"></a>Documentação da Base de Dados SQL

A Base de Dados SQL do Azure é uma base de dados relacional como serviço que utiliza o Motor do Microsoft SQL Server. A Base de Dados SQL é uma base de dados de elevado desempenho, fiável e segura que pode utilizar para criar aplicações e sites com base em dados na linguagem de programação da sua preferência, sem ter de gerir infraestrutura. Saiba como utilizar a Base de Dados SQL com os nossos inícios rápidos, tutoriais e amostras.

## <a name="5-minute-quickstarts"></a>Inícios rápidos de cinco minutos
