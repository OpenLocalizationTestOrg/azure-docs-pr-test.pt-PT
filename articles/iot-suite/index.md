# <a name="iot-suite-documentation"></a>Documentação do IoT Suite

Tenha uma introdução rápida ao Microsoft Azure IoT Suite. Utilize as soluções pré-configuradas e acelerar o desenvolvimento de Olá da sua solução Internet das coisas (IoT). Adicione novos dispositivos (e ligue outros já existentes) através de SDKs de dispositivos para várias plataformas, incluindo Linux, Windows e sistemas operativos em tempo real. Dimensionar desde alguns toomillions de sensores de dispositivos ligados em simultâneo e facilmente confiar na disponibilidade global da Olá do Azure&mdash;, independentemente de como grandes ou pequenos seu projeto.
