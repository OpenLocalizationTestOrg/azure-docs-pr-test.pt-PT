# <a name="azure-operational-security-checklist"></a>Lista de verificação de segurança operacionais do Azure
Implementar uma aplicação no Azure é rápido, fácil e económica. Antes de implementar a aplicação na nuvem em produção útil ter uma lista de verificação para ajudar a avaliar a aplicação relativamente a uma lista de ações de segurança operacional essenciais e recomendados para a ter em consideração.

## <a name="introduction"></a>Introdução

O Azure oferece um conjunto de serviços de infraestruturas que pode utilizar para implementar as suas aplicações. Segurança operacionais do Azure refere-se os serviços, controlos e funcionalidades disponíveis para os utilizadores para proteger os seus dados, aplicações e outros recursos no Microsoft Azure.

-   Para obter a vantagem máxima fora da plataforma de nuvem, recomendamos que tiram partido dos serviços do Azure e siga a lista de verificação.
-   As organizações que investem recursos avaliar a preparação operacional das respetivas aplicações antes de iniciar e tempo ter uma taxa muito superior de satisfação que quem não. Quando executar este trabalho, listas de verificação podem ser um mecanismo valiosas para se certificar de que as aplicações são avaliadas de forma consistente e de forma holística.
-   O nível de avaliação operacional será varia consoante o nível de maturidade da organização na nuvem e a fase de desenvolvimento da aplicação, precisará de disponibilidade e requisitos de confidencialidade de dados.

Também pode ser utilizado para o ajudar a criar uma estratégia de migração e a operação de nuvem segura para a sua organização.
