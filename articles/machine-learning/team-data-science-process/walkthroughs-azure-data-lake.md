# <a name="azure-data-lake-data-science-walkthroughs-using-u-sql"></a>Instruções de ciência de dados do Azure Data Lake utilizando U-SQL

Estas instruções utilizam U-SQL com o Azure Data Lake para realizar a Análise Preditiva. Siga os passos descritos no processo de ciência de dados de equipa. Para obter uma descrição geral do processo de ciência de dados de equipa, consulte [o processo de ciência de dados](overview.md). Para uma introdução ao Azure Data Lake, consulte [descrição geral do Azure Data Lake Store](../../data-lake-store/data-lake-store-overview.md).

Instruções de ciência de dados adicionais que executar o processo de ciência de dados do agrupamento estão agrupadas pelo **plataforma** que utilizam. Consulte [instruções ao executar o processo de ciência de dados de equipa](walkthroughs.md) para um itemization destes exemplos.


## <a name="predict-taxi-tips-using-u-sql-with-azure-data-lake"></a>Prever sugestões taxi utilizando U-SQL com o Azure Data Lake

O [utilização do Azure Data Lake de ciência de dados](data-lake-walkthrough.md) instruções mostram como utilizar o Azure Data Lake para fazer a exploração de dados e tarefas de classificação binária de uma amostra do NYC taxi conjunto de dados para prever ou não uma sugestão é paga por um cliente.
