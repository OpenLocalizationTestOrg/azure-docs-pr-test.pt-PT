# <a name="how-toostop-being-billed-for-azure-remoteapp"></a>Como a ser cobrados toostop para o Azure RemoteApp
> [!IMPORTANT]
> O Azure RemoteApp vai ser descontinuado a 31 de agosto de 2017. Olá leitura [anúncio](https://go.microsoft.com/fwlink/?linkid=821148) para obter mais detalhes.
> 
> 

Desde que tenham coleções do Azure RemoteApp, mesmo com sem utilizadores atribuídos, será faturado do RemoteApp. toostop este faturação, eliminar todas as coleções. 

toodelete a coleção:

1. No portal clássico do Azure Olá, abra a página de RemoteApp Olá.
2. Clique em coleção Olá pretende toodelete e, em seguida, clique em **eliminar** em Olá parte inferior da página Olá.
3. Repita esta para cada coleção na sua subscrição. 

Poderá demorar alguns minutos toodelete cada coleção.
